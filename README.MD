Este repositório existe para prática deliberada em computação. A ideia é implementar, do zero, algoritmos, estruturas de dados, técnicas de IA e outros fundamentos que fazem parte da formação de um cientista da computação. O objetivo é consolidar conhecimento pela via prática, revisitar tópicos clássicos e manter um acervo organizado para consulta futura.

A implementação dos mesmos algoritmos em diferentes linguagens ajuda a entender melhor modelos mentais, custos, limitações e estilos de programação. Ao longo do tempo, esse material deve servir como referência para pesquisa, estudos avançados e possíveis atividades de docência.

Todo o conteúdo ficará aberto para facilitar compartilhamento, revisão e evolução contínua. O foco é construir uma base sólida que acompanhe minha trajetória como profissional e acadêmico.
